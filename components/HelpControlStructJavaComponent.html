<div class="help-content">
  <h1>Control Structures (Java/C++/C# Style)</h1>

  A control structure determines the order in which statements will be executed. In addition to sequential execution, the Jeroo language supports four control structures
  <ul>
    <li> if</li>
    <li> if-else</li>
    <li> cascaded if (a variation on if-else)</li>
    <li> pretest while</li>
  </ul>
  <br>
  <br>
  <h3>The<i> if </i>Structure</h3>
  The <i>if</i> structure is used to define an optional block of code.
  <br>
  <br>

  <table border="1" cellpadding="5">
    <tr>
      <td>
        <font size="3" color="#000000" face="courier">
          <b>if(</b> <i>condition</i> <b>)<br>
            &#123;</b><br>
          <pre>    // true block  </pre>
          <b>&#125;</b>
          <br>
          <pre>// next statement</pre>
        </font>
  </table>

  <ul>
    <li> If the condition is true, execute the "true block " exactly once,<br>
      and continue with the "next statement".</li>
    <li> If the condition is false, skip the "true block", <br>
      and resume execution with the "next statement".</li>
    <li> Do not go back and recheck the condition.</li>
  </ul>
  <br>
  <br>

  <h3>The<i> if-else</i> Structure</h3>
  The <i>if-else</i> structure is used to define alternate blocks of code. <br>
  Only one of the blocks will be executed.
  <br>
  <br>

  <table border="1" cellpadding="5">
    <tr>
      <td>
        <font size="3" color="#000000" face="courier">
          <b>if(</b> <i>condition</i> <b>)<br>
            &#123;</b><br>
          <pre>    // true block  </pre>
          <b>&#125;<br>
            else<br>
            &#123;</b><br>
          <pre>    // false block  </pre>
          <b>&#125;</b><br>
          <pre>// next statement</pre>
        </font>
  </table>

  <ul>
    <li> If the condition is true, execute the "true block " exactly once, <br>
      and resume execution with the "next statement".</li>
    <li> If the condition is false, execute the "false block" exactly once,<br>
      and resume execution with the "next statement".</li>
    <li> Do not go back and recheck the condition.</li>
  </ul>
  <br>
  <br>

  <h3>The <i>cascaded if</i> Structure</h3>
  The <i>cascaded if</i> is a variation on the if-else structure. <br>
  The <i>cascaded if</i> is used to define three or more blocks of code. <br>
  Only one of the blocks will be executed.
  <br>
  <br>

  <table border="1" cellpadding="5">
    <tr>
      <td>
        <font size="3" color="#000000" face="courier">
          <b>if(</b> <i>condition-0</i> <b>)<br>
            &#123;</b><br>
          <pre>    // block-0  </pre>
          <b>&#125;<br>
            else if(</b> <i>condition-1</i> <b>)<br>
            &#123;</b><br>
          <pre>    // block-1  </pre>
          <b>&#125;</b><br><br>
          <br>
          // other else-if blocks go here<br>
          <br>
          <br>
          <b>else if(</b> <i>condition-n</i> <b> )<br>
            &#123;</b><br>
          <pre>    // block-n  </pre>
          <b>&#125;<br>
            else<br>
            &#123;</b><br>
          <pre>    // default block  </pre>
          <b>&#125;</b><br>


          <pre>// next statement</pre>
        </font>
  </table>

  <ul>
    <li> Jeroo checks the conditions in the order that they appear.</li>
    <li> When the first true condition is located, the corresponding block will be executed,
      and all other blocks and conditions will be skipped.</li>
    <li> If all conditions are false, the default block will be executed.</li>
    <li> After a block has been executed, execution resumes with the "next statement".</li>
  </ul>
  <br>
  <br>
  <h3>The<i> pretest while </i>Structure</h3>
  The <i>pretest while</i> structure (also called a while loop) is used to define a block of code that will be executed repeatedly as long as a specified condition is true.
  <br>
  <br>

  <table border="1" cellpadding="5">
    <tr>
      <td>
        <font size="3" color="#000000" face="courier">
          <b>while(</b> <i>condition</i> <b>)<br>
            &#123;</b><br>
          <pre>    // loop body  </pre>
          <b>&#125;</b>
          <br>
          <pre>// next statement</pre>
        </font>
  </table>

  <ul>
    <li> When the condition is true, execute the "loop body " once,<br>
      then go back and reevaluate the condition.</li>
    <li> When the condition is false, skip the "loop body", <br>
      and resume execution with the "next statement".</li>
  </ul>
</div>
